# Find the Guilty

🕵️‍♂️ **Find the Guilty** es un juego móvil de detectives inmersivo con estética de cómic y narrativa interactiva.

## Descripción

En este juego, asumes el papel de un investigador privado encargado de resolver crímenes complejos. Interactúa con sospechosos, busca pistas y toma decisiones que afectarán el curso de la historia.

## Características Principales

*   **Estética de Cómic:** Visuales estilizados que recuerdan a las novelas gráficas clásicas.
*   **Narrativa Ramificada:** Tus decisiones importan y llevan a diferentes desenlaces.
*   **Minijuegos:** Mecánicas interactivas como descifrado de códigos y forzado de cerraduras.
*   **Gestión de Inventario:** Recolecta y examina pruebas clave.

## Tecnologías Utilizadas

*   [React Native](https://reactnative.dev/)
*   [Expo](https://expo.dev/)
*   [TypeScript](https://www.typescriptlang.org/)
*   [Zustand](https://github.com/pmndrs/zustand) para gestión de estado.
*   [Lucide React Native](https://lucide.dev/) para iconos.

## Instalación y Ejecución

1.  **Clonar el repositorio:**
    ```bash
    git clone https://github.com/danybarreto/find-the-guilty.git
    cd find-the-guilty
    ```

2.  **Instalar dependencias:**
    ```bash
    npm install
    # o
    yarn install
    ```

3.  **Ejecutar el proyecto:**
    ```bash
    npx expo start
    ```
